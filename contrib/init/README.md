Sample configuration files for:
```
SystemD: tokyocoind.service
Upstart: tokyocoind.conf
OpenRC:  tokyocoind.openrc
         tokyocoind.openrcconf
CentOS:  tokyocoind.init
macOS:   org.tokyocoin.tokyocoind.plist
```
